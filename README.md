Aurora Login is a modern and responsive login interface built entirely with HTML, CSS, and JavaScript. The project showcases a clean glassmorphism-inspired design with an animated aurora background, floating CSS elements, interactive form validation, password visibility toggle, ripple button effects, mouse-follow glow, and smooth animations. Developed without using any external frameworks, libraries, images, or icon packs, it demonstrates core frontend development skills while maintaining a lightweight and visually engaging user experience across desktop and mobile devices.

LIVE DEMO: https://aurora-login-page.netlify.app/
